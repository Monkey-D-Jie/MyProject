此目录主要是记录泛型擦除这一特性的特点。
泛型擦除的主要目的其实就是为了：向低版本兼容。
但这种‘暴力式’的兼容也会带来一些弊端。主要是以下的几个。
1）泛型参数不支持基本类型，只支持引用类型（因为泛型最终会被“擦除”为基本类型）。
2）运行时只能对原始类型进行检测。
3）不能实例化类型参数；
4）不能实例化泛型数组；
